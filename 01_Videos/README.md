# Video Projects

Each video project should stay minimal.

```text
V001_短标题/
└── 短标题_script.docx
```

The Word document is the source of truth for each script.

Only add extra files when they are actually needed:

- `exports/`: final video exports or captions.
- `analytics.md`: only after the video is posted and needs a written review.

Codex should not create brief/script/post-copy Markdown files by default.
